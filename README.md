Incident Data Analysis & Insights Dashboard
Project Overview

This project is an end-to-end data analytics application built using Python and Streamlit to analyze incident data and uncover meaningful insights.

Tech Stack

Python

Pandas, NumPy

Matplotlib

Streamlit

Features

Interactive dashboard with filters (Region, Injury Type, Gender)

KPI metrics (Total Incidents, Most Affected Region, Dominant Injury Type)

Time-series analysis of incidents

Injury type and demographic analysis

Region and district-level insights

Data Cleaning & Preprocessing

Removed duplicate records

Handled missing values using median and default categories

Standardized categorical variables

Converted date column to datetime format

Key Insights

Certain regions show consistently higher incident rates

Specific injury types dominate across datasets

Majority of incidents involve non-hostile participants

Age distribution highlights key affected demographic groups

How to Run
streamlit run app.py
